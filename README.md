# SCIO Independent Doctor API

### To Setup

```bash
npm i
```

### To Run On Dev
```bash
node app
```

OR

```bash
npm run dev
```

OR can run `app.js` directly from `nodemon` if using live reload.


# To Run on prod
```bash
npm start
```

OR

```bash
node start
```

### To Run Test

```bash
npm test
```
